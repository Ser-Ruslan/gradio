# Сервер логистической регрессии на Gradio

## Описание

Сервер с пользовательским интерфейсом на Gradio для демонстрации модели логистической регрессии.

## Структура проекта

- `logistic_regression.py` - Класс `LogisticRegressionModel` с методом `predict`
- `gradio_logistic_app.py` - Gradio сервер для логистической регрессии
- `test_logistic_regression.py` - Тесты для функции предсказания
- `requirements.txt` - Зависимости проекта

## Установка и запуск

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Запуск тестов:
```bash
python test_logistic_regression.py
```

3. Запуск сервера:
```bash
python gradio_logistic_app.py
```

Сервер будет доступен по адресу: http://localhost:7861

## Модель логистической регрессии

Формула предсказания:
```
p = 1 / (1 + exp(-z))
где z = b0 + b1*x1 + b2*x2
```

Где:
- b0 = -10.0 (свободный член)
- b1 = 0.2 (коэффициент для оценки по ИС)
- b2 = 0.15 (коэффициент для баллов по Python)
- x1 = оценка по ИС
- x2 = количество баллов по Python

Класс 1: вероятность > 0.5
Класс 0: вероятность ≤ 0.5

## Функции

### LogisticRegressionModel
- `predict(input_array)` - Возвращает класс (0 или 1)
- `predict_proba(input_array)` - Возвращает вероятность класса 1
- `predict_with_plot(input_array)` - Возвращает предсказание и графики

## Тесты

Тесты проверяют:
1. Предсказание класса 1 (высокая вероятность)
2. Предсказание класса 0 (низкая вероятность)
3. Диапазон вероятностей [0,1]
4. Обработка списка вместо numpy массива
5. Граничные случаи

## Gradio интерфейс

Интерфейс позволяет вводить:
- Оценку по ИС (0-100)
- Баллы по Python (0-50)

Возвращает:
- Текстовый результат с интерпретацией
- Вероятность класса 1
- 3D график поверхности вероятностей и контурную карту с границей решения
