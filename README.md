# Сервер линейной регрессии на Gradio

## Описание

Сервер с пользовательским интерфейсом на Gradio для демонстрации модели линейной регрессии.

## Структура проекта

- `linear_regression.py` - Класс `LinearRegressionModel` с методом `predict`
- `gradio_app.py` - Gradio сервер с интерфейсом
- `test_linear_regression.py` - Тесты для функции предсказания
- `requirements.txt` - Зависимости проекта

## Установка и запуск

1. Установка зависимостей:
```bash
pip install -r requirements.txt
```

2. Запуск тестов:
```bash
python test_linear_regression.py
```

3. Запуск сервера:
```bash
python gradio_app.py
```

Сервер будет доступен по адресу: http://localhost:7860

## Модель линейной регрессии

Формула предсказания:
```
y = b0 + b1*x1 + b2*x2
```

Где:
- b0 = 48.6 (количество баллов по ИС)
- b1 = 2 (оценка по ИС)
- b2 = 45.9/50 (количество баллов по Python / 50)
- x1 = оценка по ИС
- x2 = количество баллов по Python

## Функции

### LinearRegressionModel

- `predict(input_array)` - Возвращает float или int предсказание
- `predict_with_plot(input_array)` - Возвращает предсказание и 3D график

### Тесты

Тесты проверяют:
1. Целочисленный результат предсказания
2. Дробный результат предсказания  
3. Обработку списка вместо numpy массива
4. Одномерный массив numpy

## Gradio интерфейс

Интерфейс позволяет вводить:
- Оценку по ИС (0-100)
- Баллы по Python (0-50)

Возвращает:
- Текстовый результат предсказания
- 3D график поверхности регрессии с точкой предсказания
